# Craft — The Design Studio 🏠

**Designing Interiors, Defining Elegance**

Premium interior design & 3D visualization studio website based in Morbi & Rajkot, Gujarat, India.

![Craft Design Studio](Logo.png)

## 🌐 Live Preview

Open `index.html` in your browser to view the website.

## ✨ Features

- **Awwwards-level animations** — Rotated text reveals, magnetic cursor, parallax scrolling
- **Horizontal scroll gallery** — GSAP ScrollTrigger-powered project showcase
- **Responsive design** — Optimized for all screen sizes (desktop, tablet, mobile)
- **SEO optimized** — JSON-LD schema, OpenGraph tags, semantic HTML5
- **Accessible** — WCAG-compliant focus states, skip links, ARIA labels
- **Performance optimized** — Lazy loading, deferred scripts, preloaded assets

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, CSS Grid, Flexbox, animations
- **JavaScript** — Vanilla JS (ES6+)
- **GSAP 3** — ScrollTrigger for scroll-based animations
- **Lucide Icons** — Lightweight SVG icon set
- **Google Fonts** — Cormorant Garamond, Inter, Space Grotesk
